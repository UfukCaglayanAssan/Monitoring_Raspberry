📋 Yardım

IP Adresi Seçimi:

  • Şirket ağında kullanılacak IP aralığında olmalı
  • Diğer cihazlarla çakışmamalı
  • Router'ın DHCP aralığı dışında olmalı

Yaygın IP Aralıkları:

  • 192.168.1.x - Ev/Ofis ağları
  • 192.168.0.x - Ev ağları
  • 10.0.0.x - Kurumsal ağlar
  • 172.16.x.x - Kurumsal ağlar

Güvenlik Notları:

  • IP değişikliği sonrası sistem yeniden başlar
  • Yeni IP ile erişim sağlanmalıdır
  • Port 5000 üzerinden erişim devam eder

⚠️ Önemli Uyarı

IP konfigürasyonu değiştirildiğinde sistem otomatik olarak yeniden başlatılacak ve yeni IP adresi üzerinden erişim sağlanacaktır. Lütfen yeni IP adresini not alın.
